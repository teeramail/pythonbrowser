Firefox Thermal Printer Setup

Follow these steps to configure Firefox for your 58mm thermal printer:

Step 1: Open Print Dialog

Click the button below to open the print dialog:

Step 2: Configure Print Settings

In the print dialog that appears:

  1. Select ThermalPrinter as your printer
  2. Click More settings to expand options
  3. Set Paper size to Custom
  4. Enter 58 for Width and 50 for Height
  5. Select Millimeters as the unit
  6. Set Margins to None
  7. Set Scale to 100% (not "Fit to page")
  8. Uncheck Print headers and footers
  9. Uncheck Print backgrounds (optional)

Step 3: Save Settings

Important: Click Save to save these settings for future use.

Name the preset something like "58mm Thermal Printer"

Step 4: Test Print

After saving your settings, click Print to test if your settings work correctly.

You should get a small receipt with the text "ทดสอบการพิมพ์" printed on your thermal printer.

Troubleshooting

  • If paper still feeds too much, try decreasing the height value (e.g., to 40mm)
  • If text is cut off, try increasing the width slightly
  • If Firefox doesn't remember your settings, you may need to save them as a preset each time
