58mm Thermal Printer Configuration

Firefox Print Settings

Step 1: Open Print Dialog

Click this button to open Firefox's print dialog:

Step 2: Configure Print Settings

  1. Select ThermalPrinter from the printer list
  2. Click More settings to expand options
  3. Under Paper size, select Custom
  4. Enter 58 for Width and 50 for Height
  5. Make sure Millimeters (mm) is selected
  6. Set Margins to None
  7. Set Scale to 100%
  8. Uncheck Print headers and footers

Step 3: Save Your Settings

Click the Save button to create a preset named "58mm Thermal Printer"

Important: This will save your settings for future use!

Step 4: Print Test Page

Click Print to test your settings.

You should get a small receipt with Thai text.

Troubleshooting

  • If paper feeds too much, try reducing the height (e.g., 40mm)
  • If text is cut off, try increasing the width slightly
  • If Firefox doesn't remember settings, save them as a preset each time
  • If printing fails, check that the printer is properly connected and has paper
